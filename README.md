# Puzzling ECG System

## Project Overview
The **Puzzling ECG** project aims to design, build, and test a functional electrocardiogram (ECG) system The primary goal is to accurately capture and display electrical heart signals by applying electronic design principles and signal processing techniques. This project was developed as part of the Engineering Practice and Design unit to cultivate practical laboratory skills and technical communication.

## Mission Statement
Our purpose is to produce a reliable and safe system capable of identifying key heart signal components, specifically **P, QRS, and T waves**, while maintaining a stable and clear waveform through iterative hardware and software improvements.

*   **Goal 1:** Design and assemble the functional ECG circuit (End of Project Week).
*   **Goal 2:** Implement signal conditioning (amplification/filtering) for a stable waveform (Tuesday).
*   **Goal 3:** Achieve a readable trace with identifiable P, QRS, and T waves (Wednesday).

## Constraints & Challenges
*   **Time:** A strict 5-day window to design, test, and refine the system.
*   **Environment:** Significant electrical noise and interference impacting signal quality.
*   **Safety:** Strict requirements for working with bio-signals to ensure user safety.

## Team Culture & Values
The project is guided by a commitment to **Respect, Collaboration, Accountability, and Continuous Improvement**. Technical decisions are made through group discussion and evidence-based testing.
